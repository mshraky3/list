# دليل شركات التقنية — التدريب الصيفي والتعاوني (نشر Vercel)

هذا المجلد يحتوي على الملفات اللازمة لنشر دليل شركات التقنية على منصة Vercel.

## خطوات النشر:

1. **تثبيت Vercel CLI (اختياري):**
   إذا كنت تريد النشر من خلال سطر الأوامر:
   ```bash
   npm i -g vercel
   ```

2. **تسجيل الدخول والنشر:**
   داخل هذا المجلد (`tech-companies-sa`) قم بتشغيل الأمر التالي:
   ```bash
   vercel
   ```
   واتبع التعليمات التي تظهر لك.

3. **النشر عبر GitHub (الموصى به):**
   - قم بإنشاء مستودع (Repository) جديد على GitHub.
   - ارفع محتويات هذا المجلد فقط إليه.
   - اذهب إلى موقع [Vercel](https://vercel.com/) وقم بربط المستودع لنشر الموقع تلقائياً.

## محتويات المجلد:
- `index.html`: الملف الرئيسي للدليل (تم نسخه من `Tech_Companies_Final_AR.html`).
- `vercel.json`: ملف إعدادات النشر لمنصة Vercel.
